# Specification

## Summary
**Goal:** Add member discovery and follower system to enable users to browse all members, follow/unfollow users, and view follower/following lists.

**Planned changes:**
- Create Members page displaying all registered users with profile pictures, usernames, and follower counts
- Implement backend follower/following system with follow, unfollow, and relationship query functions
- Add Follow/Unfollow button to user profile pages with real-time count updates
- Display follower and following counts on profiles with clickable links to detailed lists
- Create Followers and Following list pages showing user cards with follow status
- Filter reels feed to prioritize content from followed users
- Add Members navigation link in header

**User-visible outcome:** Users can browse all members, follow and unfollow other users, view follower/following lists, and see a feed that prioritizes content from people they follow.
